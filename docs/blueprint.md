# **App Name**: TechTradeHub Academy

## Core Features:

- User Authentication: User authentication with Firebase (Google, X, Email/Phone) for secure access to courses and progress tracking.
- Course Library: Browse and filter courses by category (Futures Trading, Web3, Crypto, Tech Skills, AI & Machine Learning) and level (Beginner, Intermediate, Advanced).
- Course Pages: Course pages display title and explanatory text. Bundle discounts applied to course category.
- AI Chatbot Assistant: AI chatbot "Tech Gee" will provide automated answers to student questions.
- Course Progress Tracking: Real-time tracking of student progress based on time spent on content and logged via Firebase.
- Certificate Generation: Final assessment will check student graduation with downloadable PDF certificate.
- Admin Dashboard: Admin users will have access to a secure dashboard (post-Firebase login) to manage all web content and user access.

## Style Guidelines:

- Dark Gray (#1E1E1E or #2A2A2A) – Used as the main page background to create a modern, focused learning environment, reducing eye strain.
- Teal (#00C4B4) – Applied to buttons, links, and highlights to draw attention to interactive elements and convey trust and innovation.
- Green (#00FF00) – Utilized for success indicators (e.g., checkmarks, completed tasks) to signify achievement and positivity.
- Blue (#007BFF) – Incorporated in navigation elements and secondary buttons to suggest reliability and guidance.
- White (#FFFFFF) – Primary text color for high contrast against dark backgrounds, ensuring readability.
- Light Gray (#B0B0B0) – Used for placeholders, disclaimers, and subtle details.
- Orange (#FF6200) – Added for call-to-action buttons or social media highlights to inject energy and urgency.
- Graph/Chart: Represents "Track your learning progress" – Placed in the features section next to progress-related text, colored Teal (#00C4B4).
- Star/Badge: Represents "Earn verifiable certificates" – Positioned in the features or course section, colored Teal or White.
- Checkmark: Indicates completion – Used next to completed tasks or features in lists, colored Green (#00FF00).
- Magnifying Glass: Represents "Smart Search" – Placed in the search bar or course library filter, colored Yellow (#FFCA28) or White.
- Speech Bubble: Represents messaging/chatbot – Located in the chatbot UI or features section, colored Green or White.
- Book Stack: Represents the course library – Positioned next to the course section title, colored White.
- Robot: Represents "Tech Gee" chatbot – Placed as a floating icon in the bottom-right corner, colored Blue (#1E90FF) or White.
- User (Silhouette): For username/full name fields – Inside input fields on the left, colored Light Gray (#B0B0B0).
- Envelope: For email fields – Inside input fields on the left, colored Light Gray.
- Lock: For password fields – Inside input fields on the left, colored Light Gray.
- Social Media Icons (Google, Facebook, Twitter): For social logins – Placed below login/signup forms or in the footer, using brand colors (Red #DB4437, Blue #4267B2, Light Blue #1DA1F2).
- Back Arrow: For navigation – Located in the top-right corner of the header, colored Blue with a White arrow.
- Downward Arrow: For scroll prompts – Positioned at the bottom of sections, colored White.
- Primary Button: Purpose: Call-to-action (e.g., "Enroll Now," "Login"). Design: Teal background (#00C4B4), White text, 5px border-radius, 0.5rem padding, hover effect with 10% opacity reduction. Placement: Hero section, course cards, login form.
- Secondary Button: Purpose: Navigation or less urgent actions (e.g., "Learn More," "Back"). Design: Blue background (#007BFF), White text, 5px border-radius, hover effect with 10% opacity increase. Placement: Header navigation, course details.
- Success Button: Purpose: Indicate completion (e.g., "Mark Complete"). Design: Green background (#00FF00), White text, 5px border-radius, hover effect with slight scale-up (1.05). Placement: Progress tracking, module end.
- Outline Button: Purpose: Subtle actions (e.g., "Forgot Password"). Design: White border (#FFFFFF), Transparent background, White text, 5px border-radius, hover effect with Teal border. Placement: Login/signup form.
- Icon-Only Button: Purpose: Minimal actions (e.g., chatbot toggle). Design: Circular (50px diameter), Blue background (#1E90FF), White icon (e.g., Robot), hover effect with 10% opacity reduction. Placement: Bottom-right corner for chatbot.
- Disabled State: Design: Gray background (#6C757D), Light Gray text, no hover effect. Placement: Buttons for unavailable features.
- Responsive Sizing: Minimum 44px touch target for mobile, adjustable for desktop with mouse precision.
- Implementation of consistent font choices for headings and body text to ensure readability and align with the platform's tech-oriented theme.
- Smooth transitions and animations (e.g., CSS transitions for toggles, scroll-triggered effects) to ensure a seamless user experience across all page sections.
